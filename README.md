# Installation

## Installing dependency

### Python modules
`pip install -r requirements.txt`

### HTS Engine & Open JTalk

Install hts_engine API first before installing Open JTalk

- https://hts-engine.sourceforge.net/
- https://open-jtalk.sourceforge.net/

## Start the servers

```
cd server
./tools/start-JPL.sh
```

## Stop the servers

```
cd server
./tools/stop-JPL.sh
```
